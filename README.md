### git github代码托管
git init <br>
git add README.md 代码改后要写入暂存区<br>
git commit -m "first commit" 添加到本地仓库<br>
git branch -M main 本地分支<br>
git remote add origin https://github.com/hiSpace0/community.git 远程主机名<br>
git push -u origin main
